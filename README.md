## ⚙️ How to Run a Project

1️⃣ **Clone this repository**

```bash
git clone https://github.com/YOUR-USERNAME/25-reactjs-interview-projects.git

2️⃣ Navigate to any project folder

cd project-name

3️⃣ Install dependencies

npm install

4️⃣ Run the project

npm run dev
